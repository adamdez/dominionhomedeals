Section 13 - Motivated Seller PPC Conversion Principles

The non-negotiables for this category:

Distress angle = ad group = landing page = form. One promise per funnel. A foreclosure searcher never sees a generic "we buy houses" page. A landlord never sees inherited copy. Message match is 60% of the conversion rate fight in this category.

Speed beats polish. A motivated seller in distress is calling 3-5 buyers. The investor who responds in <5 min wins disproportionately. Your form-to-call SLA matters more than your headline.

Friction kills. One-question first form ("What's the property address?") then progressive disclosure. Multi-field forms above the fold cut conversion rate by 40-60% in this category.

Trust signals are required, not optional. Distressed sellers are paranoid about scams. Real photo of the owner (you), real phone number, BBB or Google review screenshots, "we've bought X houses in Spokane since [year]," and at least one testimonial with a face. Stock photos are conversion poison.

Phone secondary, form primary - for now. You said it: nobody calls anymore. Build the form funnel right and you'll get 70-85% of leads via form. Phone call tracking is a Phase 2 problem.

Negative keywords are the entire game at small budgets. At $20/day, every wasted click is 8-15% of your daily budget. Aggressive negatives aren't optional - they're the difference between profitable and dead.

Geographic discipline. Spokane County only, Presence-only targeting. The biggest budget waste at small spend is paying for clicks from people who aren't in your buy-box.

TASK 5 - GCLID capture in hidden form field

Add to the head of every page (or in your global JS):

<script>
(function() {
  function getParam(name) {
    var url = new URL(window.location.href);
    return url.searchParams.get(name);
  }
  var gclid = getParam('gclid');
  if (gclid) {
    try {
      localStorage.setItem('gclid', gclid);
      localStorage.setItem('gclid_ts', Date.now().toString());
    } catch(e) {}
  }
  document.addEventListener('DOMContentLoaded', function() {
    var stored = null;
    try { stored = localStorage.getItem('gclid'); } catch(e) {}
    if (!stored) return;
    var forms = document.querySelectorAll('form');
    forms.forEach(function(f) {
      if (f.querySelector('input[name="gclid"]')) return;
      var input = document.createElement('input');
      input.type = 'hidden';
      input.name = 'gclid';
      input.value = stored;
      f.appendChild(input);
    });
  });
})();
</script>

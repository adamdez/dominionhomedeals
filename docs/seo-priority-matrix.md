# Dominion SEO Priority Matrix

Scoring notes:
- `Impact`: estimated ranking/business upside for Dominion, not a universal Google weight.
- `Effort`: `S`, `M`, `L`.
- `Channel`: `On-site`, `GBP`, `Off-site`, or `Mixed`.
- `Status`: `Done`, `Ready now`, `Needs external asset`, or `Needs platform access`.

## Where returns fall off

- `1-8`: core money zone
- `9-15`: still very worth doing
- `16-22`: worth doing once the core is strong
- `23-30`: do if easy, broken, or part of a compounding stack

## Immediate build order

1. Finish Spokane location consistency everywhere public.
2. Strengthen `/sell`, `/sell/as-is`, and service-area architecture with better internal linking and structure.
3. Add breadcrumbs/schema to key pages.
4. Run a real speed audit and fix the largest mobile bottlenecks.
5. Start collecting real reviews and public case-study assets.
6. Push local citations and links once business data is locked.
